## Set up

Follow: https://www.haskell.org/downloads

I chose minimum installer and installed directly to my local machine (macOS). I use Stack for package management.


## Play in ghci

```sh
ghci

# (compile and) load a file
:load <file>
:l <file>
:l basic # basic.sh

# reload the modules
:reload
:r
```
